## BitCollector

**데이터 파일(/data)**
1. `btc.csv`: 비트코인 csv파일
2. `etc.csv`: 이더리움클래식 csv파일
3. `eth.csv`: 이더리움 csv파일
4. `xrp.csv`: 리플코인 csv파일
5. `qtum.csv` : 퀀텀 csv파일
6. `bch.csv` : 비트코인캐시 csv 파일


`bitcollector.py`: keras, tensorflow Backend RNN Python File
